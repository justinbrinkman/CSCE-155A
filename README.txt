CSCE 155A Gradebook Calculator

Two imports are required to run the program:
import java.util.*;
import java.text.DecimalFormat;

Files:
1. PSEUDOCODE.txt -> Contains pseudocode for writing the program in any language
2. Gradebook1.java -> Source file that contains code that can be compiled and run in a Java IDE
3. README.txt -> Contains descriptions for files related to the program
4. TEST.txt -> Contains the printed results of 2 program test trials
